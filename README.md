命令行下的微信客户端

[urwid](https://github.com/urwid/urwid) (Console user interface library for Python) 绘制界面

[itchat](https://github.com/littlecodersh/ItChat) (A complete and graceful API for Wechat) 访问微信网页版API

## how to install 

```
pip install TWchat -U
```

## how to use

```
twchat
```

命令行会出现二维码，扫码登录

``` 
ALT+q  exit twchat
```
